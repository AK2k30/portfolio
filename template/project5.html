Short introduction about this project

Click to see demo of project

- Hand gesture-based app opening project using OpenCV

- Utilizes computer vision techniques to detect and recognize hand gestures.

- OpenCV library employed for real-time video processing and hand tracking.

- Implements hand gesture recognition algorithms to identify specific gestures.

- Maps recognized gestures to corresponding app launching actions.

- Can be extended to include additional functionality beyond app opening, such as controlling media playback or system commands.
